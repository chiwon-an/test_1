1️⃣ 사전 준비
필수 설치

Node.js 18 이상 (LTS 권장)

설치 확인:

node -v
npm -v


❗ Node.js가 설치되어 있지 않으면 프로젝트 실행이 불가능합니다.

2️⃣ 프로젝트 실행 준비
① 압축 해제

전달받은 프로젝트 압축 파일을 원하는 위치에 압축 해제합니다.

michelin_ver_1/

② 의존성 설치

프로젝트 루트 디렉토리에서 아래 명령어 실행:

npm install


처음 실행 시 1~2분 정도 소요될 수 있습니다.

3️⃣ 로컬 서버 실행
개발 모드 실행
npm run dev


정상 실행 시 콘솔 출력 예시:

▲ Next.js 16.x.x
- Local: http://localhost:3000


브라우저에서 아래 주소로 접속:

http://localhost:3000
